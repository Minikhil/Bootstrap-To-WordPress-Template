Remote Desktop Client

The Remote Desktop Connection client is used to connect to KVS.

  • Windows 7

  • Click Start => All Programs => Accessories => Remote Desktop Connection

  • Windows 8

  • Press the Windows Button on your keyboard and type "Remote Desktop Connection" then press enter.

  • Mac OS

  • Download the Mac RDP Client.

  • iOS (iPhone/iPad)

  • Download the iOS RDP Client.

Saving Reports
To save reports on your computer you will need the Remote Desktop Client (see above).

  1. Open the Remote Desktop Client

  2. Click Options, and then click the Local Resources tab

  3. Click Disk Drives, and then click Connect.
